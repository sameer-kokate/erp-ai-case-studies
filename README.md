## 📊 Case Study: Automated Custom Tariff Number Classification using AI

### 🏭 Client Context
A discrete manufacturing organization handling large volumes of imported components faced challenges in accurately determining custom tariff (HSN/HS Code) classifications during procurement and import processes.

---

### ⚠️ Problem Statement
- Manual classification of custom tariff numbers was time-consuming and error-prone  
- Incorrect classifications led to compliance risks and potential financial penalties  
- Dependency on domain experts created bottlenecks in procurement workflows  
- No standardized system existed for consistent classification across materials  

---

### 💡 Solution Approach

Designed and implemented an AI-powered classification solution by integrating a ChatGPT-based connector within the ERP system.

#### Key Components:
- Integration of AI (ChatGPT) with ERP via API connector  
- Input parameters: material description, specifications, and category  
- AI-driven classification logic to suggest appropriate tariff codes  
- Validation layer for user review and approval before final submission  

---

### ⚙️ Implementation

- Configured ERP workflow to trigger AI classification during material creation and procurement  
- Developed structured prompts to ensure accurate and consistent tariff suggestions  
- Enabled fallback logic for ambiguous classifications requiring manual validation  
- Integrated output into procurement and compliance processes  

---

### 📈 Results & Impact

- ⏱️ Reduced manual effort in tariff classification by ~70%  
- 🎯 Improved classification accuracy and consistency across materials  
- ⚡ Faster procurement processing with reduced dependency on experts  
- ✅ Enhanced compliance with customs regulations  
- 📊 Standardized classification approach across multiple business units  

---

### 🧠 Key Learnings

- AI can significantly enhance ERP decision-making when integrated with structured workflows  
- Prompt design plays a critical role in achieving reliable AI outputs  
- Human-in-the-loop validation is essential for compliance-critical processes  

---

### 🚀 Future Enhancements

- Integration with customs databases for real-time validation  
- Continuous learning model based on historical classification decisions  
- Expansion to automate other compliance-related processes
